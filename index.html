Group API

Divides people into groups in various ways.

Routes

/team returns a full list of team members

/groups/{ size of group } returns groupings of team members into the specified group size. Uneven groupings will result in remaining team members sharing a smaller group.

/roundrobin returns sequences of users, where each user is followed by another without conflicts.

/new adds a team member, provide a teamMember parameter in the request body.

/combine adds an array of team members, send an array, teamMembers, in the request body.

/delete/{ team member name } removes a team member from the list of team members.
